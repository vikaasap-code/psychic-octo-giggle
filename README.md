# AstroMatch - Приложение знакомств на основе астрологии

**AstroMatch** - это iOS приложение для знакомств, которое использует анализ натальных карт для определения совместимости между пользователями.

## 🌟 Особенности

### Астрологический анализ
- **Генерация натальных карт** на основе даты, времени и места рождения
- **Анализ совместимости** между пользователями по множественным критериям
- **Визуализация натальных карт** с интерактивными элементами
- **Детальный анализ аспектов** между планетами партнеров

### Система совместимости
- Анализ совместимости элементов (огонь, земля, воздух, вода)
- Солнце-Луна совместимость для эмоциональной связи
- Венера-Марс анализ для романтических отношений
- Меркурий анализ для коммуникации
- Синастрические аспекты между картами

### Пользовательский интерфейс
- **Swipe-интерфейс** для просмотра потенциальных партнеров
- **Карточки пользователей** с астрологической информацией
- **Система матчинга** с мгновенными уведомлениями
- **Чаты с астрологическими советами** для улучшения общения

## 🏗 Архитектура

### Модели данных
- `NatalChart` - натальная карта пользователя
- `CompatibilityResult` - результат анализа совместимости
- `UserProfile` - профиль пользователя с астрологическими данными
- `Match` - информация о совпадениях между пользователями

### Сервисы
- `AstrologyCalculator` - расчет позиций планет и натальных карт
- `CompatibilityAnalyzer` - анализ совместимости между картами
- `UserManager` - управление состоянием пользователей и матчей

### Экраны
- `DiscoveryView` - поиск и свайп потенциальных партнеров
- `MatchesView` - список ваших матчей
- `ChatsView` - чаты с астрологическими советами
- `ProfileView` - профиль с натальной картой
- `NatalChartView` - детальная визуализация натальной карты
- `CompatibilityDetailView` - подробный анализ совместимости

## 🎯 Астрологические расчеты

### Планеты и знаки
- Поддержка всех 12 знаков зодиака
- Расчет позиций всех классических планет
- Определение ретроградности планет
- Анализ домов гороскопа

### Аспекты
- Соединение (0°)
- Оппозиция (180°)
- Трин (120°)
- Квадрат (90°)
- Секстиль (60°)
- Квинконс (150°)

### Совместимость
Алгоритм совместимости учитывает:
- Элементный баланс (40% веса)
- Солнце-Луна отношения (25%)
- Венера-Марс притяжение (20%)
- Коммуникационные аспекты (10%)
- Синастрические аспекты (5%)

## 🚀 Запуск проекта

### Требования
- iOS 15.0+
- Xcode 13.0+
- Swift 5.5+

### Установка
1. Клонируйте репозиторий
```bash
git clone https://github.com/yourusername/astromatch.git
```

2. Откройте проект в Xcode
```bash
cd astromatch
open Package.swift
```

3. Установите зависимости и запустите проект

### Зависимости
- SwiftUIX - расширения для SwiftUI

## 📱 Использование

### Создание профиля
1. Введите дату, время и место рождения
2. Добавьте фотографии и описание
3. Приложение автоматически рассчитает вашу натальную карту

### Поиск партнеров
1. Просматривайте карточки других пользователей
2. Изучайте показатели совместимости
3. Делайте лайки понравившимся профилям

### Общение
1. При взаимной симпатии открывается чат
2. Используйте астрологические советы для лучшего общения
3. Изучайте детальный анализ совместимости

Приложение использует темную тему с фиолетово-розовой цветовой схемой, создающей мистическую атмосферу. Интерфейс построен на SwiftUI с современными анимациями и плавными переходами.

- Основной: фиолетовый (#6B46C1)
- Акцент: розовый (#EC4899)
- Фон: темный с градиентами
- Текст: белый/серый для контраста

# AstroMatch - Приложение для знакомств на основе натальных карт

## Описание
AstroMatch - это iOS приложение, которое помогает людям находить совместимых партнеров на основе анализа их натальных карт. Приложение использует астрологические принципы для определения совместимости между людьми.

## Основные функции
- 📱 **Профили пользователей** с астрологическими данными
- 🔮 **Анализ совместимости** на основе натальных карт
- 💫 **Детальный астрологический отчет** для каждой пары
- 🎯 **Поиск совместимых партнеров** по астрологическим критериям
- 📊 **Визуализация совместимости** с помощью графиков и диаграмм
- 💬 **Чат между совместимыми пользователями**

## Технологии
- **iOS 15.0+**
- **SwiftUI** для современного UI
- **Core Data** для локального хранения данных
- **Astrology API** для расчетов натальных карт
- **Firebase** для backend и аутентификации

## Установка и запуск
1. Откройте `AstroMatch.xcodeproj` в Xcode
2. Выберите симулятор iPhone
3. Нажмите Run (⌘+R)

## Требования
- Xcode 14.0+
- iOS 15.0+
- Swift 5.7+

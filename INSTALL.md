# Инструкции по установке и запуску AstroMatch

## Требования

- **macOS** 12.0 или новее
- **Xcode** 14.0 или новее
- **iOS Simulator** или физическое устройство с iOS 15.0+

## Установка

### 1. Клонирование репозитория

```bash
git clone <repository-url>
cd AstroMatch
```

### 2. Открытие проекта в Xcode

```bash
open AstroMatch.xcodeproj
```

Или запустите Xcode и выберите "Open a project or file", затем выберите файл `AstroMatch.xcodeproj`.

### 3. Настройка проекта

1. В Xcode выберите проект `AstroMatch` в навигаторе
2. Выберите target `AstroMatch`
3. В разделе "Signing & Capabilities":
   - Выберите вашу команду разработчиков
   - Убедитесь, что Bundle Identifier уникален
4. В разделе "Deployment Info":
   - Убедитесь, что iOS Deployment Target установлен в 15.0

### 4. Выбор симулятора

1. В верхней части Xcode выберите симулятор iPhone (например, "iPhone 15 Pro")
2. Убедитесь, что выбрана схема "AstroMatch"

## Запуск

### Способ 1: Через Xcode

1. Нажмите кнопку ▶️ (Run) или используйте сочетание клавиш `Cmd + R`
2. Дождитесь компиляции и запуска приложения
3. Приложение откроется в симуляторе

### Способ 2: Через командную строку

```bash
# Сборка проекта
xcodebuild -project AstroMatch.xcodeproj -scheme AstroMatch -destination 'platform=iOS Simulator,name=iPhone 15 Pro' build

# Запуск в симуляторе
xcrun simctl boot "iPhone 15 Pro"
xcodebuild -project AstroMatch.xcodeproj -scheme AstroMatch -destination 'platform=iOS Simulator,name=iPhone 15 Pro' test
```

## Возможные проблемы и решения

### Ошибка компиляции

1. Убедитесь, что Xcode обновлен до последней версии
2. Очистите проект: `Product` → `Clean Build Folder` (`Cmd + Shift + K`)
3. Перезапустите Xcode

### Ошибка подписи

1. Проверьте настройки в "Signing & Capabilities"
2. Убедитесь, что у вас есть действующий сертификат разработчика
3. Попробуйте использовать "Automatically manage signing"

### Симулятор не запускается

1. Сбросьте симулятор: `Device` → `Erase All Content and Settings`
2. Перезапустите Xcode
3. Проверьте, что симулятор поддерживает iOS 15.0+
